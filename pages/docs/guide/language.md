# Language support

DeeLip's interface is English-only today, but every string in the app already runs
through a translation lookup rather than being hardcoded — the groundwork for
additional languages is already in place.

If you'd like to help bring DeeLip to another language, open a discussion or issue
on GitHub — see [Contact](/contact/).

---
Curious how any of this actually works under the hood? The engineering notes live
in [`docs/crates/i18n.md`](https://github.com/Smyrnis/DeeLip/blob/main/docs/crates/i18n.md)
on GitHub.
